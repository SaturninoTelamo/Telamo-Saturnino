Lesson And What I Have Learned

1. New material: Technology Timeline Infographics

This course explores the history of computing, from mechanical to vacuum tube to transistor. The lesson emphasises on those who made a big contribution to history, such as Charles Babbage and Douglas Engelbart. It also covers computing in a wide range of disciplines, including education, business, and healthcare. The conclusion explores how the evolution of computing will continue to influence future society and outcomes. It will teach me about the history of technology and how to use it today and how its important.

2. New material: Information Processing Cycle and Computer System

This lesson describes the basic method by which computers manage information, which includes four major stages: input, processing, storage, and output. It also emphasises the need of good information, highlighting qualities such as relevance, completeness, and correctness..

3. New material: Introduction to Computer System

introduces the four core components of a computer system: hardware, software, peopleware, and dataware. It provides a detailed explanation of each component, outlining their functions and providing relevant examples.

4. New material: Basic Concept of Computer Security

The need of protecting systems and data from misuse and unauthorised access is emphasised in this computer security lecture. It emphasises how crucial it is to safeguard personal data, classify information based on its degree of sensitivity, and be informed on common cybercrimes and their defences. The need of proactive security measures in safeguarding systems and data is emphasised in the course..

5. New material: HTML and CSS

This class covers the fundamentals of using HTML and CSS to create web pages. HTML employs tags to provide structure, while CSS styles the appearance. This lesson covers a variety of HTML tags, CSS syntax, and CSS insertion methods. It also briefly mentions JavaScript for interaction and displays wireframes and mockups for visual design. This class lays the groundwork for building basic web pages..

6. New material: The Internet and World Wide Web

In this lesson, The Internet and WWW, the various uses of the internet and its global interconnectedness are examined. It covers topics including cloud computing, machine learning, the Internet of Things, and the different ways we use the internet for communication, research, and publication. Social media is also covered in the course, along with its many forms and how they impact our lives..
